Blockchain-Based Electronic Voting System

Blockchain Ethereum Web3.js Smart Contracts

Project Overview

A secure blockchain-based voting platform developed on the Ethereum network that provides transparency, security, and immutability for electronic voting systems.

Key Features

  • Secure voter authentication and verification
  • Immutable vote recording on Ethereum blockchain
  • Transparent vote counting and verification
  • Prevention of double voting and fraud
  • Anonymous yet verifiable voting

Technologies Used

Ethereum
Web3.js
Smart Contracts
JavaScript

Impact

This project was reviewed by the Nepal Election Commission for potential future implementation, demonstrating its real-world applicability and security standards.
